# Chapter 4 | Eigenvalue Problems

in the real world (finding the roots of polynomials) and the main algorithms
involved in calculating it (Newton Iteration, etc).

## Main Focus: 
- Eigenvalues and Eigenvectors
- SVD Decomposition.
- Refresher on IVT.
- Root finding Algorithms 
    - Bisection
    - Fixed Point Iteration
    - Newton Iteration
