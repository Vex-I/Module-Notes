# Chapter 2 | System of Linear Equations
This chapter serves mostly as a refresher on the concepts of linear algebra,
before delving into further concepts in numerical analysis (not that the lecture 
is particulary rigorous in the math side, though). It focuses on the already
taught concepts of linear algebra, but mostly on the common algorithms 
pertaining to calculating matrices, mostly is pseudocode, as well as their time
complexities. It includes the basic algorithms to compute basic matrix
operations, forward elemination and backward subtitution, LU decomposition,
pivoting, as well as optimization for special systems of linear equations.
